
## To start the application
   1.  clone the repo ( git clone https://github.com/looosernk98/jet-set-go.git )
   
   2. go to jet-set-go folder( root level) and run below commands
   3.  npm install
   4.  npm start

   ## To create build
    1. npm run build

   ## To Run Test
    1. npm run test
